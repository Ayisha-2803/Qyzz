# 🏆 Rise Scholar — Ultimate Gamified Cognitive Arena

> A premium, high-fidelity gamified assessment experience styled to absolute precision. Test your cognitive bounds, accumulate XP multipliers, hold the streak flame active, and earn master credentials.

---

## 🎨 Visual Identity & Aesthetic Overview

Rise Scholar is designed with a distinctive **Cosmic Slate Theme**—moving away from generic purple gradients toward an elegant, high-contrast, glowing glassmorphism experience. 

- **Interactive Cognitive Astro-Orb**: A stunning central neural canvas on the landing page featuring animated orbits, connecting energy matrices, interactive micro-tooltips, and responsive gravity particles.
- **Micro-Animations**: Uses high-performance `motion` layouts, custom page transitions, staggered state list rollouts, and energetic win-burst celebrations.
- **Ultra-Readable Typography**: Pairs futuristic display fonts with clean mono typography for status indicators, mimicking high-tech tactical terminals.

---

## ⚡ Main Competency Engines

### 1. 🌌 The Cognitive astro-Orb Hub
An interactive visual showcase that helps players discover connected topics. Watch particles orbit concentric tracks in real-time while selecting active core nodes like the **Streak Engine**, **Master Credentials**, **Technology Core**, and the **Grand Arena**.

### 2. 📝 High-Fidelity Quiz Player
- **Tick-Perfect Timer**: Features a color-changing countdown timer that transitions from royal indigo to pulsating warning red.
- **Dynamic Progress Tracking**: Features responsive multi-step progress indicators and visual metrics with elegant slide-in entering animations.
- **Complete Keyboard/Click Layout**: Perfectly sized interactive options grids with real-time selection highlights.

### 3. 📊 Performance Decoders (Analytics)
Visualize your progress with custom-styled, interactive **Recharts** integrations:
- **Consecutive Accuracy Trend**: A filled Area Chart illustrating accuracy changes over consecutive attempts.
- **Daily XP Accumulations**: A rounded Bar Chart tracking daily XP gains.
- **Subject Tag Proficiency**: A high-contrast donut Pie Chart displaying test coverage across topics.

### 4. 🥇 Grand Arena Leaderboards
- **Natural Podiums**: Features a beautifully sculpted 1st (Center), 2nd (Left), and 3rd (Right) place visual columns layout.
- **Visual Rank Tokens**: Crown animations and custom medals highlight the top users.
- **League Roster Table**: Shows level cards, relative streaks, and live player profile specs.

### 5. 🛠️ Manual Assessment Maker & AI Registries
- Includes a robust form wizard checking for option bounds, empty values, and correct declarations.
- Designed to allow instantaneous quiz creations saved dynamically to internal databases.

## 💾 Local Storage Architecture (Ready To Play)

Rise Scholar is built with an **Offline-First Storage Engine** using persistent memory adapters. 
* Yes, **you do not need any API key or subscription to host or run this!** 
* All statistics, levels, XP multipliers, created tests, history logs, and daily challenges persist directly on the visitor's device.
* *Enterprise Capability:* A server-side connector is prepared in `server.ts` to support optional external Gemini API generation pipelines if you ever want to expand!

## 🚀 Easy Local Setup

Get your arena up and running on your device in **seconds**:

### Prerequisites
Make sure you have Node.js installed on your computer.

### Installation & Run
1. **Clone your repository**:
   ```bash
   git clone <your-github-repo-url>
   cd rise-scholar
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

🛠️ Technological Foundation
Build toolchain: Vite + React + TypeScript
Interactions / Motion: Motion (Framer Motion v11)
Analytics Rendering: Recharts (D3 Engines)
Styling framework: Tailwind CSS v4
Fluid interface iconography: Lucide React
